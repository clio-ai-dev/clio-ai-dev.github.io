A bit of what I've been doing with agents:

At work, I was handed an ambiguous requirement. Instead of spending an hour reading through our 100+ project codebase, I asked my AI agent to explain the requirement in context. It found the exact components involved and told me precisely what to implement.

I needed to propose 3 approaches for a feature. Instead of writing docs explaining each option, I spun up 3 sub-agents. Each one implemented a different approach as a separate PR. The team voted on real code, not slides.

At home, I asked an agent to provision an Azure VM, install software, and configure everything. I never touched a terminal. From there, that assistant has drafted LinkedIn posts, built a 32-concept visual guide, managed my calendar, and even built a lunar eclipse animation for my kids.

All I do is review.

People think AI coding means autocomplete. Tab, tab, tab.

Here's what AI agents are actually doing for me. Today. At work and at home.

At work

I'm a software engineer at a large tech company.

I was handed an ambiguous requirement. Instead of spending an hour reading through our 100+ project codebase, I asked my AI agent to explain the requirement in context. It found the exact components involved and told me precisely what to implement.

We spotted errors in production. I was totally new to the service. I asked the agent to connect to our log store, find the error patterns, correlate them with the codebase, and tell me what's broken. It nailed it.

My pull requests used to fail in CI because the pipeline builds differently than my local box. I added custom instructions to the repo. Now the agent runs the exact same checks locally before creating the PR. First-time passes, every time.

I used to write PR descriptions manually and respond to review comments one by one. Now the agent creates detailed descriptions based on our work together, reads the review comments, addresses them, pushes the fix, and replies.

I needed to propose 3 approaches for a feature. Instead of writing docs explaining each option, I spun up 3 sub-agents. Each one implemented a different approach as a separate PR. The team voted on real code, not slides.

I wanted to upgrade our Python toolchain but didn't trust one model's judgment. So I spun up 3 agents (Claude, Codex, Gemini) and asked them to independently figure out what the upgrade requires. They reached consensus. When the first build failed, I looped them until every issue was found.

At home

I wanted a personal AI assistant. Instead of configuring a VM myself, I asked an AI agent to connect to my Azure subscription, provision the machine, install the software, and tell me how to connect. I never touched a terminal.

From there, that assistant has:

  • Drafted 6 LinkedIn posts per week for my side business
  • Built an entire 32-concept visual guide with custom illustrations
  • Managed my calendar and created reminders for my family
  • Set up a newsletter, generated logos, wrote welcome emails
  • Built a lunar eclipse animation for my kids

All I do is review.

The shift is real

These aren't demos. This is Tuesday.

The tools went from assistants (you ask, they answer) to agents (you direct, they execute). If you're still copy-pasting from ChatGPT, you're training the wrong muscle.

This newsletter will show you how. One workflow per week. Practical. No theory.

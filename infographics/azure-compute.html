VMs

  • Full OS control needed.
  • Legacy non-container apps.
  • Compliance hardware needs.

App Service

  • Fastest path to web hosting.
  • Built-in CI/CD and SSL.
  • Focus on code, not infra.

Functions

  • Queues, timers, HTTP triggers.
  • Zero idle cost execution.
  • Rapid autoscaling for bursts.

ACI

  • Quick isolated containers.
  • CI/CD build agents/test.
  • Short-lived batch jobs.

AKS

  • Full custom networking.
  • Large-scale microservices.
  • Service mesh & GitOps.

Container Apps

  • K8s benefits, no management.
  • Dapr-based microservices.
  • Scale-to-zero efficiency.

General Tips & Cost Considerations

All services integrate with Azure DevOps and GitHub Actions. Choose VMs for maximum flexibility but higher management overhead. Use Serverless (Functions/ACA) for unpredictable workloads. AKS is the gold standard for full Kubernetes features and enterprise scale, while Container Apps offers the best middle ground for simplified microservices. Don't over-provision; start small and scale based on metrics.
